Q: Is there a Frequently Asked Questions (FAQ) page?
Yes, there is a FAQ Page.

Q: Youtube is not working?
A: Allow "Media Server" from the application list.

Q: I see data consumed by application which is blocked with AFWall+, why?
A: It's a false data. AFWall+ uses iptables which is at kernel level (lowest). Usage data shown by Android is calculated before it gets dropped by firewall.

Q: Downloads are not working with stock Browsers/Google Play Store
A: Allow "Downloads" from application list.

Q: Explain whitelist vs. blacklist
A: Whitelist is for advance users where they know which application can connect to internet including system application Blacklist is for beginners where they want to block only certain application's connecting to internet. It depends on person to person.

Q: What are all the icons on the top ?
A: LAN - Within Local wifi, Wifi, Data(2g/3g/4g), Roaming, VPN followed by revert and clear selection.
